# Knife Classification using Deep Learning

**University of Surrey | Fundamentals of Machine Learning**

## Overview

This repository contains a deep learning coursework project on fine-grained knife image classification using CNN-based architectures, data augmentation and hyperparameter tuning.

## Models Evaluated

- EfficientNet-B0
- ResNet-50
- ConvNeXt-Tiny

## Key Results

- EfficientNet-B0 test mAP: 0.449
- ResNet-50 test mAP: 0.444
- ConvNeXt-Tiny test mAP: 0.678

## Experiments

- Backbone comparison
- Data augmentation experiments
- Learning rate tuning
- Batch size tuning
- Training and test log analysis

## Repository Contents

- Assignment report
- Training logs
- Test logs
- Experimental results
